# Image-Encryption-and-Decryption-tool

This tool is used to Encrypt and Decrypt images in many different ways.

1- Rail Fence Cipher

2- Image to Text

3-Both (the Rail Fence Cipher and Text)



                                            This tool is build using PYTHON
